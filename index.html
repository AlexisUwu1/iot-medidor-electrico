Sistema IoT — Medidor de Consumo Eléctrico

Aplicación web para administrar, controlar y monitorear circuitos eléctricos simulados mediante API REST.

¿De qué trata esta aplicación?

Este sistema simula una solución IoT compuesta por múltiples circuitos eléctricos. Cada circuito puede activarse o desactivarse, cambiar su nivel de carga y generar mediciones de voltaje, corriente, potencia y energía.

La información se gestiona mediante una API REST en MockAPI y el sistema está dividido en tres módulos principales: Administración, Control y Monitoreo.

Administración

Permite crear, editar, listar y eliminar los circuitos IoT. Aquí se definen los límites de potencia y el estado inicial de cada dispositivo.

Ir a Administración

Control

Permite encender o apagar cada circuito mediante interruptores, modificar su carga (baja, media o alta) y generar lecturas manuales.

Ir a Control

Monitoreo

Visualiza el comportamiento de los circuitos en tiempo real mediante gráficas y una tabla con los últimos estados registrados, actualizados cada 2 segundos.

Ir a Monitoreo
Sistema desarrollado con Bootstrap 5 y MockAPI como backend simulado.
